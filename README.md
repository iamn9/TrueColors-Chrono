TRUE COLORS
True Color is a game inspired by the Stroop effect.  The goal of this game is to select the name of the color written that matches with the color displayed¬¬. React faster than your opponents and select the correct answers within the time limit to get points. The player with the higher number of points wins.

For more info, visit [True Color by Aurélien Hubert](https://play.google.com/store/apps/details?id=com.aurelhubert.truecolor&hl=en) on Google Play.
